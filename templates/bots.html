📞 Communications Division

Discord and Telegram bots for each project. Send commands, receive agent reports, and queue messages when offline. Configure bot tokens, toggle channels on/off, and monitor the message feed.

📖 Shoe Phone Setup Guide
🎮 Discord Bot Setup
  1. Go to discord.com/developers
  2. New Application → name it (e.g. "Max - ProjectName")
  3. Bot tab → Create Bot → Copy the token
  4. Enable Message Content Intent under Privileged Intents
  5. OAuth2 → URL Generator → select bot scope
  6. Add permissions: Send Messages, Read Message History
  7. Open the generated URL to invite bot to your server
  8. Right-click the channel → Copy Channel ID
  9. Paste token + channel ID below
✈ Telegram Bot Setup
  1. Open Telegram, search for @BotFather
  2. Send /newbot and follow the prompts
  3. Copy the bot token BotFather gives you
  4. Send any message to your new bot
  5. To get your Chat ID, send /start to the bot
  6. The bot will reply with your Chat ID
  7. Paste token + chat ID below
